# 🏠 Flat Rent Calculator

A Python project to calculate how much each person in a flat should pay, including rent, food, and electricity.

## Features
- Input rent, food, electricity units, and charges
- Adjust for amounts already paid
- Calculate total expense and per-person share

## Skills Learned
- Python basics
- Input/Output handling
- Conditional statements
- Problem solving

## How to Run
```bash
python rent_calculator.py
